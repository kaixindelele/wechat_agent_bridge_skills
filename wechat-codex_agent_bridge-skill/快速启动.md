# wechat-codex_agent_bridge-skill 快速启动

## 1. 前置条件

- 已安装 Node.js 18+
- 已安装并登录 `codex` CLI
- 已完成微信 ClawBot / ilink 绑定

```bash
codex login
```

## 2. 准备配置

```bash
cd wechat-codex_agent_bridge-skill/templates
cp bridge.config.example.json bridge.config.json
```

编辑 `bridge.config.json`，至少设置：

- `cwd`: Codex 执行任务的项目根目录
- `codexPath`: `codex` 可执行文件路径，默认 `codex`
- `sandboxMode`: `read-only` / `workspace-write` / `danger-full-access`

## 3. 安装依赖

```bash
npm install
```

## 4. 微信扫码登录

```bash
npm run setup
```

执行后会生成 `credentials.json`。

## 5. 启动桥接

```bash
npm start
```

## 6. 微信里常用命令

```text
/help
/status
/model
/model gpt-5.4
/model clear
/sandbox
/sandbox workspace-write
/stop
/clear
/send /tmp/result.txt
```

## 7. 说明

- 图片会作为 `codex exec --image` 输入给 Codex。
- 文件、语音、视频会保存到本地，再把路径告诉 Codex 去读取。
- 追问会写入工作区根目录 `.bridge-followup.md`，Codex 可以在任务执行过程中主动读取。
- 该版本不支持微信逐条批准工具调用；如需更严格控制，请把 `sandboxMode` 设为 `read-only` 或 `workspace-write`。
